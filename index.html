Difference between local storage & session storage

  • Local storage store data permanent until deleted via JavaScript or manually. But Session storage store only duration of session time.
  • Local storage capacity is 5/10mb. Session storage capacity 5mb
  • Local storage is not session based but session storage session based
  • Local storage is client side and session storage is server side
  • Session storage is best for anythings sensitive or secure in nature.

How does JavaScript work?

JavaScript is client side scripting language. It's working way is interesting.

JavaScript is single threaded and interpreted that means it executed line by line. It run on client machine inside web-browsers but must be supported by browser. Nowadays every modern web browser support JavaScript and have their engine. Google Chrome has its own engine called V8. Safari have JavaScript Core, Firefox have SpiderMonkey, Edge have Chakra.

In engine, first step is parser which is check JavaScript code for syntactic errors.In short, it parses JavaScript code. After checking code it creates the data structure called AST (it stands for Abstract Syntax Tree). Then it conversion to machine code.
